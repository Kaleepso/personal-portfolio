# Eddie Ndegwa Portfolio

This is my personal portfolio website, built using **React**, **TypeScript**, and **Vite**. It showcases my skills, projects, and experience as a passionate BBIT student at Strathmore University with a deep interest in software development and technology.

## Features
- Modern, responsive design
- Animated sections and interactive UI components
- Contact links for GitHub, LinkedIn, and email
- Built with React functional components and hooks
- Styled using Tailwind CSS

## Technologies Used
- React
- TypeScript
- Vite
- Tailwind CSS

## Getting Started
To run the project locally:

```bash
npm install
npm run dev
```
